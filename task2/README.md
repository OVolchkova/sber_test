# Задание 2. Telegram-бот по электроэнергетике РФ

Telegram-бот, реализующий систему вопрос-ответ на базе GigaChat API с RAG-системой по документу СО ЕЭС (СиПР ЭЭС России 2025–2030 гг.) и калькулятором

ССЫЛКА НА БОТА: @energy_research_bot

## Архитектура

Бот работает по схеме ReAct-агента: GigaChat сам решает, нужно ли
обратиться к базе знаний, произвести вычисления, или
ответить из общих знаний. После каждого вызова инструмента LLM получает
наблюдение и продолжает рассуждение

### Содержание файлов

`bot.py` - телеграмм бот с состояниями: ожидание API ключа, диалог
`agent.py` - агент поверх GigaChat
`rag.py` - Загрузка PDF, 
`tools.py` - инструменты калькулятора и поиска по документу
`prompts.py` - системный промпт
`build_index.py` - скрипт первичной подготовки FAISS-индекса

### Скачивание PDF и построение индекса

python build_index.py

качает СиПР-документ
(`https://www.so-ups.ru/.../sipr_ups_2025-30_fin.pdf`), создает векторные представления и сохраняет FAISS-индекс в `./vector_store/`.

Эмбеддинги считаются локально HuggingFace-моделью `intfloat/multilingual-e5-base`

### Запуск бота

python bot.py

В Telegram отправьте боту `/start`. Бот попросит API-ключ GigaChat,
после валидации перейдёт в режим диалога.
Команда `/reset` очищает историю диалога (но сохраняет API-ключ),
команда `/key` позволяет сменить API-ключ.
